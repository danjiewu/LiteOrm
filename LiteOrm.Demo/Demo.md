# DEMO 演示格式说明

## 📋 演示结构

每个演示按照以下结构进行输出：

1. **演示标题** - 用 Box 风格的分隔符突出显示
2. **📋 场景说明** - 清晰描述本演示要展示的场景
3. **📝 代码实现** - 展示使用的 C# 代码
4. Expr内容展示
5. 序列化和反序列化演示
6. **🔍 执行的 SQL** - 显示实际执行的 SQL 语句（从 `SessionManager.Current.SqlStack.Last()` 获取）
7. **✅ 查询结果** - 如果有查询的返回结果则显示

## 程序逻辑
1. 插入User、Department、SalesRecord
2. 更新演示，所有发货日期为空的电脑发货日期设为订购日期的10天以后，且不早于明天                                                    
3. 按Lambda、Expr、ExprString三种方式进行查询演示：
   查询1：所有经理按年龄从大到小排序取前10条
   查询2：2026年1月销售记录排前10的人员
